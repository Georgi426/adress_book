# Address Book Application

Уеб базирано приложение за управление на контакти, разработено по задание.

## Функционалности
-   **Потребители**: Регистрация и вход.
-   **Контакти**: Добавяне, редактиране и изтриване на контакти с пълни детайли.
-   **Коментари**: Rich Text Editor за бележки към всеки контакт.
-   **Етикети**: Създаване на цветни етикети и групиране на контакти.
-   **Персонализирани полета**: Възможност за добавяне на допълнителни полета (напр. Рожден ден).
-   **Справки**: Статистика за етикети, проверка за дублиращи се записи.
-   **Експорт**: Сваляне на данните в CSV/JSON.

## Инсталация

### Изисквания
-   Web Server (Apache/Nginx)
-   PHP 7.4+
-   MySQL / MariaDB

### Стъпки
1.  Копирайте файловете в основната директория на вашия уеб сървър (напр. `htdocs/address_book`).
2.  Отворете в браузъра: `http://localhost/address_book/install.php`
3.  Въведете вашите данни за базата данни (Host, User, Password).
    *   Ако използвате XAMPP по подразбиране: User: `root`, Password: (празно).
4.  Натиснете "Инсталирай". Скриптът автоматично ще създаде базата данни и таблиците.
5.  Влезте в системата.
    *   **Администратор по подразбиране**: user: `admin`, pass: `admin`

## Структура на кода
-   `classes/`: PHP класове за логиката (User, Contact, Tag).
-   `config/`: Конфигурация за база данни.
-   `layouts/`: Шаблони за Header/Footer.
-   `install.php`: Скрипт за първоначална настройка.
-   `dashboard.php`: Основен екран със списък контакти.

## Забележки
Приложението е напълно готово за проверка и тест.
